# Тайны космоса — деплой на Netlify

Этот небольшой статический сайт состоит из `index.html` и `logic.js`. Чтобы разместить сайт бесплатно на Netlify — есть несколько простых способов:

1) Быстро: Drag & Drop
- Открой https://app.netlify.com/drop
- Перетащи в окно папку с файлами (`index.html` и `logic.js`) или ZIP со содержимым
- Netlify сгенерирует публичный URL сразу

2) Через GitHub (рекомендуется для обновлений)
- Создай новый репозиторий на GitHub и запушь файлы
- В Netlify -> New site -> Import from Git -> выбери GitHub репозиторий -> Deploy site
- Каждый push будет автоматически деплоить сайт

3) Через Netlify CLI
- Установи CLI (если ещё не установлен):

```bash
npm install -g netlify-cli
```

- Залогинься и инициализируй/deploy:

```bash
netlify login
# один раз
netlify init    # или netlify link если уже есть сайт
netlify deploy --prod --dir="."   # публикует текущую папку
```

Дополнительно
- `netlify.toml` добавлен для минимальной конфигурации (publish = ".").
- Если вы хотите использовать `main.html` вместо `index.html`, переименуйте `index.html` в `main.html` и создайте `index.html` с редиректом, либо настройте `netlify.toml` для переадресации.

Проверка локально
- Для быстрой локальной проверки можно запустить простой HTTP-сервер:

```bash

python3 -m http.server 8000
# затем открой http://localhost:8000
```
