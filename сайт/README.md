# Сайт про программирование

Минимальный статический сайт для заметок и примеров кода.

Как запустить локально

- Откройте файл `index.html` в браузере (двойной клик) — сайт полностью статический.
- Для локальной разработки можно запустить простой HTTP-сервер, например с Python:

```bash
# из папки /home/mr_proper/Рабочий стол/сайт
python3 -m http.server 8000
# затем открыть http://localhost:8000
```

Добавление статьи

- Создайте HTML-файл в папке `posts/`, можно скопировать `posts/sample-post.html` как шаблон.
- В `index.html` добавьте ссылку на статью в секцию "Последние статьи".

Дальше можно:

- Перенести сайт на генератор (Eleventy / Next.js) для удобной работы с Markdown/MDX.
- Добавить систему сборки (npm, Tailwind, etc.).
- Подключить деплой на GitHub Pages или Vercel.

Markdown-статьи

Теперь сайт поддерживает простые Markdown-статьи. Чтобы просмотреть статью, используйте шаблон `posts/post.html` с параметром `src`, например:

```text
posts/post.html?src=sample-post.md
posts/post.html?src=hackers.md
```

Файлы Markdown хранятся в папке `posts/`.

Если хотите, могу:
- Перевести проект на Next.js + MDX и добавить поддержку подсветки и примеров.
- Подготовить шаблон статьи в Markdown.
